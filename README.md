# vue-chrome-extension-boilerplate

Boilerplate for Chrome extension using Vue.js and Webpack with auto-reload enabled.

## Scripts

```json
// install dependencies
yarn install

// build extension and watch for changes
yarn dev

// build extension zip
yarn build

// lint all source files
yarn lint
```
